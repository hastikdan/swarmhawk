SwarmHawk Documentation

SwarmHawk is a domain security intelligence platform. Enter any domain and get a full threat picture in under 60 seconds — SSL health, DNS misconfigurations, email spoofing risks, open ports, CVEs, breach exposure, dark-web credentials, IaC file leaks, and more. Results are scored 0–100 and summarised by AI into plain-English remediation steps.

This knowledge base covers every feature, check, and integration. It is updated whenever a new feature ships.

Quick Start

You can use SwarmHawk in three ways:

  1. Public scanner — Go to swarmhawk.com, type any domain, press Enter. No account needed. Returns a free preview (subset of checks).
  2. Dashboard — Log in, add your domains under Domains → Add Domain. SwarmHawk scans automatically and emails you monthly PDF reports.
  3. API / MCP — Use the REST API or MCP server to embed SwarmHawk into your own tools, scripts, or AI assistants (Claude, Cursor, etc.).

Tip: The fastest way to assess a domain you don't own is the public scanner — no login, instant results. For ongoing monitoring of your own domains, use the dashboard.

All 22 Security Checks

Every scan runs 22 independent checks in parallel. Each check returns a status (ok / warning / critical / error) and a score_impact (0–25 penalty points). The final risk score is the sum of all penalties, capped at 100.

Network & Infrastructure

🔒 SSL / TLS FREE
Verifies the SSL certificate is valid, checks expiry date, and confirms the issuer. Detects expired, self-signed, or misconfigured certificates.
● Critical — expired or invalid cert (impact 15–20)
● Warning — expires within 30 days (impact 8)
● OK — valid, issuer confirmed
↪ HTTPS Redirect FREE
Checks whether plain HTTP requests are redirected to HTTPS. A missing redirect means traffic can be intercepted in plaintext.
● Warning — site served over HTTP with no redirect (impact 10)
● OK — HTTP 301/302 to HTTPS
🛡 Security Headers FREE
Checks for HSTS, Content-Security-Policy, X-Frame-Options, X-Content-Type-Options, and server version disclosure.
● Warning — missing headers (impact 5–20 cumulative)
● OK — all four headers present
🌐 DNS FREE
Verifies domain resolves correctly and checks for basic DNS misconfigurations. Foundation check — if DNS fails, most other checks will also error.
● Critical — domain does not resolve (impact 20)
● OK — resolves cleanly
⚡ Response Time FREE
Measures HTTPS response time. Slow responses indicate DDoS, overloaded servers, or misconfigured CDN. Also catches HTTP 5xx server errors.
● Critical — >10s or HTTP 5xx (impact 10–15)
● Warning — 5–10s (impact 5)
📡 Shodan FREE
Queries Shodan for the domain's IP. Returns open ports, known CVEs, service banners (nginx version, OpenSSH, etc.), and risk tags. Uses the paid Shodan API for full results; falls back to free InternetDB if unavailable.
● Critical — CVEs detected or dangerous ports open (impact 20)
● Warning — >10 open ports (impact 8)
🚪 Open Ports FREE
Active socket scan of 12 high-risk ports with a 2-second timeout: FTP (21), SSH (22), Telnet (23), SMTP (25), MySQL (3306), Postgres (5432), Redis (6379), MongoDB (27017), RDP (3389), VNC (5900), HTTP-alt (8080/8443).
● Critical — database ports or Telnet/RDP/VNC open (impact 20)
● Warning — SSH, FTP, or SMTP exposed (impact 8)
🔍 IP Intelligence PAID
Resolves the domain to its IP, then queries ip-api.com for ISP, ASN, and flags: TOR exit node, known proxy/VPN, bulletproof hosting, or datacenter. Rate limited to 45 req/min on the free tier of ip-api.com.
● Critical — TOR exit node or known anonymizer (impact 20)
● Warning — bulletproof/high-abuse hosting (impact 8)

Threat Intelligence

🦠 URLhaus FREE
Checks abuse.ch URLhaus — a real-time malware URL database. Detects domains currently serving malware, ransomware droppers, or exploit kits. No API key needed.
● Critical — active malware URLs live (impact 25)
● Warning — historical malware, now offline (impact 10)
📧 Spamhaus DBL FREE
DNS-based lookup against the Spamhaus Domain Block List. Identifies spam domains, phishing infrastructure, malware C&C servers, and botnet controllers. Completely free — no API key needed.
● Critical — listed as spammer/phishing/malware/botnet (impact 20–25)
● Warning — abused legitimate domain (impact 10–12)
💻 Google Safe Browsing FREE
Checks if Chrome's Safe Browsing database has flagged this domain for malware, phishing, or unwanted software. Requires a free Google API key set via GOOGLE_SAFEBROWSING_KEY.
● Critical — actively flagged by Chrome (impact 25)
● OK / Skipped — clean or no API key set
🧪 VirusTotal FREE
Queries 70+ antivirus and threat intelligence engines via the VirusTotal API. Returns malicious/suspicious engine counts, reputation score, and category tags. Requires VIRUSTOTAL_API_KEY.
● Critical — 5+ engines: malicious (impact 25)
● Warning — 1–4 engines flagged (impact 6–12)
💥 Breach Exposure FREE
Queries HaveIBeenPwned for breaches linked to this domain. Identifies how many data breach databases the domain has appeared in and names the breach sources.
● Critical — domain found in 1+ breaches (impact 15)
● OK — not found in any breach database
🐛 CVE Scan FREE
Detects software versions from HTTP response headers and body (server, X-Powered-By, generator meta tags), then looks up known CVEs in the NVD database. Higher rate limits with NVD_API_KEY.
● Critical — known CVEs found in detected software (impact 15–25)
● Warning — outdated software, no known CVEs yet (impact 8)
🕵️ Dark Web Leaks FREE
Scans dark-web leak databases and Telegram channels via ParanoidLab for credential dumps, infostealer logs, ransomware group mentions, and combo lists linked to the domain. Requires PARANOIDLAB_API_KEY.
● Critical — ransomware leak or infostealer logs (impact 20–25)
● Warning — Telegram mentions or credential records (impact 5–10)
🎭 Typosquat FREE
Generates 200+ lookalike domain permutations (character swaps, omissions, doublings, homoglyphs, TLD variations, hyphen insertions) and checks DNS resolution for each. Detects active phishing infrastructure impersonating the target.
● Critical — 3+ typosquat domains registered (impact 10)
● Warning — 1–2 lookalike domains found (impact 5)

Application Security

💻 SAST — Source Exposure FREE
Probes for exposed source code and debug endpoints: /.git/HEAD, /.git/config, /phpinfo.php, /?XDEBUG_SESSION_START=1, /server-status, and config samples. A live /.git/HEAD means the entire source repository is downloadable.
● Critical — git repo or phpinfo exposed (impact 20–25)
● Warning — composer.json / package.json accessible (impact 10)
📦 SCA — Dependency CVEs FREE
Fetches exposed dependency manifests (/package.json, /composer.json, /requirements.txt, /Gemfile.lock) and scans for known-vulnerable library versions — log4j, lodash 4.17.20, axios 0.21.0, jQuery 1.x/2.x, and more.
● Critical — vulnerable dependency detected (impact 15)
● Warning — manifest accessible, no known CVEs (impact 8)
🌐 DAST — App Testing FREE
Passive crawl of common sensitive paths: /admin, /wp-admin, /phpmyadmin, /.env, /backup.zip, /debug, /console, /api/v1. Detects unauthenticated admin panels and exposed credential files.
● Critical — .env with credentials or unauth admin panel (impact 20)
● Warning — admin login page exposed (impact 10)
🏗 IaC — Config Exposure FREE
Checks for exposed infrastructure-as-code and deployment files: /.env, /docker-compose.yml, /Dockerfile, /terraform.tfstate, /k8s.yaml, /ansible.cfg, /deploy.sh. Terraform state files contain plaintext secrets.
● Critical — .env or terraform.tfstate accessible (impact 25)
● Warning — Dockerfile or docker-compose exposed (impact 12)

OSINT & Identity

📋 WHOIS / RDAP FREE
Queries the RDAP protocol (rdap.org) for domain registration date, expiry, and registrar. Newly registered domains (<30 days) are a strong phishing signal. Privacy-masked registrations on business domains are flagged.
● Critical — domain <30 days old (impact 15)
● Warning — expires in <30 days or privacy-masked (impact 5–8)
✉️ Email Security FREE
Queries DNS TXT records for SPF (Sender Policy Framework) and DMARC. Missing SPF means anyone can send email as your domain. DMARC p=none means spoofed emails are still delivered — only monitored, not blocked.
● Critical — no SPF record (impact 12)
● Warning — no DMARC, or DMARC p=none (impact 6–8)

AI Threat Summary

Every paid scan includes an AI-generated threat briefing powered by Claude. It contains:

  1. Executive summary — one paragraph, plain English, suitable for board or legal team
  2. Critical findings — the highest-risk issues with context on why they matter
  3. Threat scenarios — realistic attack paths an adversary could exploit
  4. Prioritised recommendations — ordered by effort/impact ratio, with estimated remediation time
  5. Intelligence notes — hosting provider, CDN, DDoS protection context

Note: The AI summary is generated once per scan and cached. Re-scanning a domain generates a fresh summary reflecting the latest findings.

Frequently Asked Questions

Is SwarmHawk legal to use on any domain?

Yes. SwarmHawk uses only passive OSINT — it queries publicly available data sources (DNS, RDAP, public APIs, Shodan's existing index). It does not perform active exploitation, credential brute-forcing, or any technique that requires authorization. You can safely run it on any domain.

How long does a scan take?

A full 22-check scan typically completes in 20–45 seconds. Checks run in parallel. The slowest checks are open port scanning (socket timeouts, ~24s worst case) and the AI summary generation (~10s). The result appears progressively as checks complete.

What does "error" status mean?

An error status means the check could not complete — usually a network timeout, a missing API key (VirusTotal, Safe Browsing), or a DNS resolution failure. It carries a small 5-point penalty but is not a confirmed security finding. Check the detail field for the exact reason.

Why is my risk score high when the site looks fine?

Risk score reflects security configuration, not visual appearance. Common causes of elevated scores: missing DMARC (email spoofing risk), missing security headers (browser-side protection gaps), typosquat domains registered by third parties, or historical breach exposure. Use the AI summary to understand which findings are most urgent for your context.

Can I white-label or embed SwarmHawk?

Yes — the API supports embedding scan results in your own dashboards or client portals. The MSP plan includes branded PDF report generation. Contact us for white-label enquiries.

How often is the Shodan data refreshed?

SwarmHawk queries the Shodan API live on each scan — results reflect Shodan's most recent crawl of that IP, typically within the past 1–7 days. The scan does not use cached Shodan data.

What does "locked" mean on a check?

Locked checks are available to paid subscribers only. The check ran and produced a result, but the result is hidden for free accounts. Upgrade to see all 22 check results.
